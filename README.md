# RApidez

RApidez is a planning project for an institutional web platform designed to improve communication and organization within a school environment.

The goal of the system is to centralize several administrative and academic processes that are traditionally handled through physical media or inefficient procedures.

## Project Purpose

The platform aims to solve common organizational issues detected within the institution, such as:

- Printer reservation requests
- Computer borrowing management
- School announcements and communications
- Administrative document requests
- Access to academic information

By integrating these functions into a single web platform, the system seeks to improve efficiency, communication and accessibility for both students and teachers.

## Proposed Technologies

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- PHP

Database:
- MySQL

## Development Approach

The project planning considered the use of modern software development practices including:

- Scrum methodology for iterative development
- Version control using Git
- Client–server architecture

## Context

RApidez was developed as part of a technical high school ICT program. The project focuses on applying web development knowledge to create practical technological solutions for real problems within an educational institution.
